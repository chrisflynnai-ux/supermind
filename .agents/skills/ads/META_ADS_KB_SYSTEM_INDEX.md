## SYSTEM OVERVIEW

The Meta Ads KB System is a comprehensive, modular knowledge base for Meta advertising campaign management, creative production, measurement, diagnostics, and scaling. Built from 7 NotebookLM compendiums synthesizing expertise from 8+ industry experts (Sam Piliero, Professor Charley T, Chase Chappell, Dr. Matt Shiver, etc.).

**Core Paradigm:** Andromeda (Creative IS Targeting) - Meta's algorithm matches users to creatives based on content analysis, making manual targeting largely obsolete.

### 6. KB-PERFORMANCE-DIAGNOSTICS v1.0 (45KB)
**File:** `KB-META-ADS-PERFORMANCE-DIAGNOSTICS-v1.0.xml`
**Purpose:** 15 failure modes, 7 anti-patterns, troubleshooting workflows
**Status:** ✅ Production Ready

**15 Failure Modes (Full Playbooks):**
1. **Creative Fatigue** - Frequency >3.0, Hook Rate drops >30%
2. **Attribution Hallucination** - Platform ROAS high, MER flat
3. **The Spend Hog** - One ad gets 90% of CBO budget
4. **Learning Limited Loop** - <50 conversions/week per ad set
5. **Pixel Malnutrition** - Zero conversions despite sales occurring
6. **Hook-Body Mismatch** - High hook rate, low hold rate (clickbait)
7. **Lag Time Disconnect** - Sales cycle >7 days, pixel blind
8. **Audience Fragmentation** - Too many overlapping ad sets
9. **The Junk Lead Spiral** - Low CPL, zero backend conversions
10. **Cost Cap Freeze** - Campaign stops spending (bid too low)
11. **Variable Testing Trap** - Testing variations not concepts
12. **Retargeting Saturation** - Frequency >10, CPMs astronomical
13. **Native Format Misfit** - Ads look too "commercial"
14. **Front-End Obsession** - Optimizing for CTR not conversions
15. **Algorithmic Whiplash** - Performance crashes after edits

**7 Anti-Patterns:**
1. Variable Testing (test concepts not variations)
2. Frequent Pausing (allow 7-day windows)
3. Front-End Obsession (optimize for conversion velocity)
4. Reactionary Pausing (evaluate over 7 days)
5. Micro-Segmentation (use Broad targeting)
6. The Perfect Ad ("ugly" ads outperform polished)
7. The Retargeting Trap (focus on new customer acquisition)

**Circuit Breakers:**
- Profit Floor: IF ncROAS <1.0 for 7 days → STOP SCALING
- Spend Spiral: IF CPA >3x Target for 48h → KILL AD
- Creative Dead End: IF 10 concepts get $0 spend → AUDIT OFFER

**Use Cases:**
- Daily diagnostic checks
- Troubleshooting campaign issues
- Weekly creative refresh protocols
- Performance degradation root cause analysis

- Pack Count: >1 per $10k spend (Rule of 1:10k)

## QUICK START GUIDE

### For New Accounts ($0-$10k/month):
1. Start with **KB-CAMPAIGN-ARCHITECTURE** → Implement basic CBO structure
2. Use **KB-ANDROMEDA-CREATIVE-ENGINE** → Create 3-5 initial concepts
3. Reference **KB-IMAGE-GENERATOR** → Produce Lazy Static batch (11 variants)
4. Monitor with **KB-IMAGE-ANALYZER** → Weekly hook rate audits
5. Calculate with **KB-MEASUREMENT-ATTRIBUTION** → Establish ncROAS baseline
6. Troubleshoot with **KB-PERFORMANCE-DIAGNOSTICS** → As issues arise
7. Scale with **KB-SCALING-STRATEGIST** Level 1 protocols once ncROAS >1.5x

### For Growth Accounts ($10k-$30k/month):
1. **KB-CAMPAIGN-ARCHITECTURE** → Implement M3 Swim Lanes + Pack System
2. **KB-ANDROMEDA-CREATIVE-ENGINE** → Weekly Creative Flywheel (Rule of 1:10k)
3. **KB-IMAGE-GENERATOR** → Nano Banana sprints (15 variants/week)
4. **KB-SCALING-STRATEGIST** → Level 2 protocols (Pack System scaling)
5. **KB-MEASUREMENT-ATTRIBUTION** → Weekly Incremental Attribution audits
6. **KB-PERFORMANCE-DIAGNOSTICS** → Daily diagnostic checks

### For Scale Accounts ($30k-$100k+/month):
1. **KB-SCALING-STRATEGIST** → Level 3 protocols (Cost Caps, Value Rules)
2. **KB-CAMPAIGN-ARCHITECTURE** → 5x5 Testing Matrix implementation
