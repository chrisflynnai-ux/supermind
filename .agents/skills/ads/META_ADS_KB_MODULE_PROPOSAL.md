## Proposed Meta Ads KB Module Structure

### **KB 1: KB-META-ADS-CAMPAIGN-ARCHITECTURE** (Base Module, ~60KB)
**Purpose:** Campaign structure foundation (M3 Swim Lanes, budget allocation, learning phase mechanics)

**Contents:**
- **M3 Swim Lane Architecture:**
  - Prospecting (Broad targeting, creative self-selects, 60-70% budget)
  - Retargeting (Site visitors 7-90d, video viewers 25%+, engagers, 20-30% budget)
  - Retention (Past customers 0-365d, win-back/upsell, 5-10% budget)
  - Exclusion logic validation

- **Campaign Structure Frameworks:**
  - 3:2:2 Flexible Ad System (low-variance testing for new campaigns)
  - 5x5 Testing Matrix (high-variance for scaling $20k+/month)
  - Learning Phase Mechanics (50 conversions/week minimum)
  - Budget allocation formulas
  - Scaling increments (20% max per adjustment)

- **Platform Architecture:**
  - Pixel + Conversion API setup
  - Attribution window configuration (7-day click / 1-day view)
  - Event tracking hierarchy (primary vs secondary conversions)
  - Placement strategy (Automatic vs Manual)

- **Campaign Naming Conventions:**
  - Format: [LANE]_[OBJECTIVE]_[CREATIVE_ANGLE]_[DATE]
  - Examples: PROS_CONV_FounderVSL_ScamHook_0119

- **Constitutional Compliance:**
  - No fake urgency or manufactured scarcity
  - Proof discipline (claims grounded in evidence)
  - Platform policy compliance (Meta Advertising Policies)

- **Dependencies:**
  - Calls: KB-META-ADS-ANDROMEDA (creative-as-targeting paradigm)
  - Calls: KB-META-ADS-MEASUREMENT (ncROAS, tracking validation)

**Consciousness Profile:**
- Target: Campaign architects, media buyers, strategists
- Neuro-Box: SMART (logical structure) + SAFE (budget risk management)

### **KB 2: KB-META-ADS-ANDROMEDA-CREATIVE-ENGINE** (~55KB)
**Purpose:** Creative-as-targeting paradigm, hook generation, concept testing frameworks

**Contents:**
- **Andromeda Retrieval Engine:**
  - Creative IS targeting (not demographics)
  - Algorithm matches users to creatives based on content analysis
  - Psychographic self-selection via hooks
  - Demographics for reporting only

- **12 Creative Frameworks:**
  1. Founder-Led VSL (Hook → Promise → Proof → Pivot → CTA, 60-90sec)
  2. Hero vs. Villain Story (External + Internal villain, stakes, plan, CTA)
  3. Lazy Static Headlines (Authentic images + text overlays, <15min production)
  4. Scam Hook Formula (Contrarian positioning, call out industry lies)
  5. 3-Step Viral Hook (Visual interrupt + Text hook + Audio cue)
  6. Nano Banana AI Static Machine (AI-accelerated scaling of winners)
  7. Post ID Harvesting (Preserve social proof when duplicating)
  8. Visual Hook Architecture (80% effort on first 3 seconds)
  9. Visual Hook Audit (Diagnostic for Hook Rate <25%)
  10. Concept Over Variation (Test different stories, not color swaps)
  11. Hook-Body Alignment (Self-selecting hooks match offer)
  12. Native Format Matching (Blend with feed, avoid "ad-like" polish)

- **Creative Flywheel:**
  - Rule of 1:10k (ship 1 new creative per $10k monthly spend)
  - Weekly creative production sprints
  - Creative bench building (always 3-5 concepts in testing queue)
  - Rotation before fatigue (14-day max lifespan for heroes)

- **Concept Diversity Validation:**
  - Test concepts (different hooks/stories/mechanisms), not variations
  - Diversity ratio calculation (unique hooks + unique mechanisms / total creatives)
  - Anti-pattern detection (same hook, different colors = variation testing)

- **Hook Rate Optimization:**
  - Target >30% Hook Rate (3-second stop scroll)
  - Target >50% Hold Rate (15-second engagement)
  - First 3 seconds determine success

- **Constitutional Compliance:**
  - No clickbait hooks (deliver on promise)
  - No sensationalist language without proof
  - Authentic/ugly beats polished (native format)

- **Dependencies:**
  - Called by: KB-META-ADS-CAMPAIGN-ARCHITECTURE
  - Calls: KB-META-ADS-IMAGE-GENERATOR (visual production)

**Consciousness Profile:**
- Target: Creative strategists, copywriters, content creators
- Neuro-Box: SPECIAL (pattern interrupts) + SMART (strategic hooks)

### **KB 4: KB-META-ADS-IMAGE-ANALYZER** (~45KB)
**Purpose:** Visual creative performance analysis, Hook Rate diagnostics, competitive research

**Contents:**
- **Visual Hook Audit Framework:**
  - Visual interrupt check (chaos/surprise/ugly vs polished)
  - Text overlay validation (curiosity gap or self-selecting)
  - Movement detection (motion in first 3sec vs static)
  - Audio presence (voice/sound vs background music only)
  - Feed contrast (pattern interrupt vs blends in)

- **Hook Rate Diagnostics:**
  - <25% = weak visual hook (audit required)
  - 25-35% = acceptable (proceed)
  - >35% = strong hook (scale candidate)
  - >40% but low conversion = hook-body mismatch

- **Performance Analysis:**
  - Hook Rate trend (detect 30%+ decline = fatigue)
  - Hold Rate analysis (15-sec+ engagement)
  - Click-through rate (trust conversion)
  - Engagement rate (shares, comments, saves)

- **Competitive Visual Research:**
  - Meta Ads Library scraping protocol
  - Competitor hook pattern extraction
  - Format analysis (static vs video, aspect ratio, text density)
  - Swipe file building (categorize by niche, hook type, visual style)

- **Native Format Analysis:**
  - Feed aesthetic matching (does it blend or scream "ad"?)
  - Authenticity score (polished vs raw/ugly)
  - Scroll-stopping elements (movement, chaos, contrast)

- **Creative Fatigue Detection:**
  - Hook Rate decline >30% from baseline
  - Frequency spike >3.5
  - CPA increase >40% from baseline
  - Delivery slowdown despite budget

- **A/B Test Design:**
  - Visual variant testing (different backgrounds, overlays, colors)
  - Text overlay split tests (hook phrasing)
  - Format tests (1:1 vs 9:16, static vs carousel)

- **Constitutional Compliance:**
  - No compliance flag triggers (sensationalism, prohibited content)
  - Authentic representation (no misleading visuals)

- **Dependencies:**
  - Called by: KB-META-ADS-ANDROMEDA-CREATIVE-ENGINE
  - Calls: KB-META-ADS-MEASUREMENT (performance data integration)

**Consciousness Profile:**
- Target: Performance analysts, creative directors, optimizers
- Neuro-Box: SMART (data-driven analysis) + SPECIAL (pattern recognition)

### **KB 5: KB-META-ADS-MEASUREMENT-ATTRIBUTION** (~55KB)
**Purpose:** ncROAS calculation, incrementality testing, attribution validation, tracking integrity

**Contents:**
- **ncROAS Framework:**
  - New Customer ROAS = new customer revenue / ad spend
  - Blended ROAS includes repeat customers (attribution hallucination)
  - Target ncROAS >1.5x for sustainable scaling
  - Attribution inflation detection (blended ROAS - ncROAS gap)

- **Incremental Attribution Methods:**
  - **Holdout Tests:**
    - Baseline period: 14-30 days ads on, measure MER
    - Holdout period: 14-30 days ads off, measure organic revenue
    - Lift calculation: (baseline revenue - organic revenue) / ad spend
  - **Conversion Lift Studies:**
    - Meta runs controlled experiment (test vs control groups)
    - Provides incremental conversion count + cost
    - Requires $50k+/month spend for statistical validity
  - **ncROAS Tracking:**
    - Segment reporting: new vs repeat customers
    - Calculate weekly/monthly for scale decisions

- **Tracking Validation:**
  - Pixel firing verification (Meta Pixel Helper Chrome extension)
  - Conversion API configuration check (server-side tracking)
  - Event match quality score (>7.0 = good)
  - Test purchase completion (verify in Events Manager within 20min)
  - Attribution window validation (7-day click / 1-day view default)

- **MER (Marketing Efficiency Ratio):**
  - Total revenue / total marketing spend (all channels)
  - Health check metric (should align with platform ROAS trends)
  - MER flat while spend increases = attribution hallucination

- **Lead Gen Lag-Time Optimization:**
  - Optimize for backend conversion (call booked, sale made), not just lead capture
  - Primary event = backend action, secondary = lead form submit
  - After 50 backend conversions, switch to backend-only optimization
  - Reduces junk leads, improves lead-to-sale conversion rate

- **Failure Mode: Pixel Malnutrition:**
  - Symptoms: No conversions tracked despite sales (verified in CRM/Stripe)
  - Diagnosis: Pixel Helper shows no firing, Events Manager empty
  - Recovery: Fix pixel implementation, set up Conversion API, extend attribution window
  - Prevention: Test purchase BEFORE launching campaigns

- **Failure Mode: Attribution Hallucination:**
  - Symptoms: Platform ROAS looks great but MER flat/declining
  - Diagnosis: Compare platform ROAS to MER (>1.5x gap likely)
  - Recovery: Run holdout test, calculate ncROAS, use for scale decisions
  - Prevention: Track ncROAS from day 1, segment new vs repeat customers

- **Constitutional Compliance:**
  - Transparent attribution (acknowledge platform vs actual lift)
  - No false claims about ROAS (use ncROAS for client reporting)

- **Dependencies:**
  - Called by: KB-META-ADS-CAMPAIGN-ARCHITECTURE, KB-META-ADS-SCALING-STRATEGIST
  - Calls: Python validation tools (calculate_ncROAS, validate_tracking_integrity)

**Consciousness Profile:**
- Target: Performance marketers, analysts, CFOs, growth teams
- Neuro-Box: SMART (data accuracy) + SAFE (budget risk management)

### **KB 6: KB-META-ADS-PERFORMANCE-DIAGNOSTICS** (~60KB)
**Purpose:** 15 failure modes with diagnostics, recovery protocols, troubleshooting playbook

**Contents:**
- **15 Failure Modes (Full Playbooks):**
  1. **Creative Fatigue** — Hook Rate drops 30%+, frequency >3.5, CPA spikes 40%+
  2. **Attribution Hallucination** — Platform ROAS great, MER flat (repeat customers inflating)
  3. **Learning Limited Loop** — <50 conversions/week, audience too narrow, budget too low
  4. **Hook-Body Mismatch** — High Hook Rate but low conversion (attracts wrong audience)
  5. **Offer-Channel Mismatch** — Good creative metrics but no conversions (cold traffic → high-ticket without nurture)
  6. **Pixel Malnutrition** — No conversions tracked despite actual sales (tracking broken)
  7. **Creative-Avatar Misalignment** — Creative performs for wrong demographic (not target ICP)
  8. **Retargeting Saturation** — Retargeting CPA spikes 100%+, frequency >5, pool exhausted
  9. **Compliance Flags** — Ad rejected for policy violations (health claims, sensationalism)
  10. **Native Format Misfit** — Low engagement, feels "ad-like" (too polished, not native)
  11. **Under-Resourced Creative Pipeline** — Can't ship weekly, bottlenecked, scaling stalled
  12. **Budget Cliff** — Profitable at $X but unprofitable at $1.5X+ (audience ceiling hit)
  13. **Misread MER** — Cutting profitable channels due to platform attribution gaps
  14. **Audience Fragmentation** — Dozens of ad sets, learning spread thin (fighting Andromeda)
  15. **Funnel Blindness** — Optimizing ads without fixing landing page conversion rate <2%

- **Playbook Structure (All 15):**
  - **Symptoms:** Observable signals (metrics, delivery status, user feedback)
  - **Root Cause:** Why it happens (mechanism failure)
  - **Diagnosis Steps:** How to confirm (specific checks, tests)
  - **Recovery:** Step-by-step fix (prioritized actions)
  - **Prevention:** How to avoid (proactive monitoring, process changes)

- **7 Anti-Patterns:**
  1. Variable Testing (testing color swaps instead of concepts)
  2. Frequent Pausing (resets learning phase)
  3. Front-End Obsession (ignoring backend LTV)
  4. Avatar Sprawl (over-segmentation by demographics)
  5. Retargeting Crutch (prospecting doesn't work, only retargeting profitable)
  6. Creative Starvation (same creatives 30+ days)
  7. Over-Segmentation (manual targeting fighting Andromeda)

- **Decision Heuristics (25-50 IF-THEN rules):**
  - **Diagnostic:**
    - IF Hook Rate <25% THEN audit visual hook using Visual Hook Audit
    - IF Hook Rate >35% but CPA high THEN hook-body mismatch
    - IF learning limited 7+ days THEN consolidate campaigns or increase budget
    - IF all creatives perform similarly THEN testing variations not concepts
  - **Optimization:**
    - IF profitable CPA achieved THEN scale budget 20% every 3-5 days max
    - IF winner emerges (CPA 30%+ better) THEN kill bottom 50%, reallocate budget
    - IF creative fatigue detected THEN ship replacement concept same day
  - **Scaling:**
    - IF spending <$10k/month THEN use 3:2:2 framework
    - IF spending $20k+/month THEN use 5x5 framework
    - IF spending $50k+/month THEN ship 5+ concepts weekly
  - **Troubleshooting:**
    - IF compliance flags received THEN audit hooks for sensationalism
    - IF pixel firing but no conversions THEN check Conversion API setup
    - IF budget unspent 50%+ THEN increase audience size or bid cap too low

- **Constitutional Compliance:**
  - No black-hat workarounds (platform violations)
  - Transparent troubleshooting (acknowledge when issues are unfixable)

- **Dependencies:**
  - Called by: All other KBs (universal troubleshooting reference)
  - Calls: KB-META-ADS-MEASUREMENT (for ncROAS/MER validation)
  - Calls: KB-META-ADS-ANDROMEDA-CREATIVE-ENGINE (for creative fixes)

**Consciousness Profile:**
- Target: Campaign managers, troubleshooters, performance analysts
- Neuro-Box: SMART (diagnostic logic) + SAFE (risk mitigation)

### **KB 7: KB-META-ADS-SCALING-STRATEGIST** (~50KB)
**Purpose:** Growth phase management, budget scaling, incrementality validation, geographic expansion

**Contents:**
- **3-Phase Scaling System:**
  - **Phase 1: Validation ($0-10k/month):**
    - Goal: Achieve ncROAS >1.5x consistently for 14+ days
    - Framework: 3:2:2 testing (low variance)
    - Creative volume: 3-5 concepts/week
    - Structure: Single M3 swim lane setup
    - Exit criteria: ncROAS >1.5x for 14+ days, MER stable

  - **Phase 2: Growth ($10k-50k/month):**
    - Goal: Scale budget 20% weekly while maintaining ncROAS >1.5x
    - Framework: 5x5 testing for prospecting
    - Creative volume: 5-7 concepts/week
    - Structure: Add 2nd prospecting campaign (different creative angles)
    - Exit criteria: MER >3.0, ncROAS >1.5x at $50k/month

  - **Phase 3: Scale ($50k-200k+/month):**
    - Goal: Expand to new markets/products while maintaining MER >3.0
    - Framework: Creative Flywheel (Rule of 1:10k = 5-20 concepts/week)
    - Creative volume: 10-20 concepts/week
    - Structure: 3-5 prospecting campaigns, geographic expansion, product line expansion

- **Budget Scaling Protocol:**
  - 20% max increase per adjustment (avoid algorithm shock)
  - 3-5 day intervals between increases (let learning stabilize)
  - Monitor: ncROAS, MER, frequency, Hook Rate, CPA trend
  - Circuit breaker: If ncROAS drops >20%, pause scaling, diagnose issue

- **Budget Cliff Detection:**
  - Symptoms: Profitable at $X spend, unprofitable when scaling to $1.5X+
  - Diagnosis: Audience size vs daily spend ratio (need 100x minimum), frequency rising fast
  - Recovery: Add new prospecting campaign with different creative angles, expand geos, test new creative concepts

- **Multi-Campaign Scaling:**
  - Avoid single campaign ceiling (algorithm hits audience limit)
  - Build 3-5 prospecting campaigns BEFORE hitting ceiling
  - Each campaign uses different creative angles (unlock different audience segments)
  - Maintain separate retargeting for each prospecting campaign (clean attribution)

- **Geographic Expansion:**
  - Tier 1 markets: US, UK, CA, AU (highest ROAS, highest CPA)
  - Tier 2 markets: EU (DE, FR, ES, IT), NZ, SG (moderate ROAS, moderate CPA)
  - Tier 3 markets: LATAM, SEA, Eastern EU (lower ROAS, lower CPA)
  - Localization requirements: Language, currency, cultural hooks, compliance

- **Incrementality Validation:**
  - When to run holdout tests: Annually, before major budget jumps ($50k → $100k+), when stakeholders question effectiveness
  - When to use Conversion Lift Studies: Spending $50k+/month, need statistically valid data
  - How to interpret lift data: True incremental ROAS, realistic scale targets, channel contribution

- **Scaling Metrics Dashboard:**
  - ncROAS trend (weekly)
  - MER trend (weekly)
  - Spend vs target (daily)
  - Creative production rate (concepts shipped per week)
  - Audience ceiling indicators (frequency, CPA trend, delivery rate)

- **Constitutional Compliance:**
  - No fake growth claims (incremental ROAS only)
  - Transparent about plateau risks (audience ceilings exist)

- **Dependencies:**
  - Called by: KB-META-ADS-CAMPAIGN-ARCHITECTURE
  - Calls: KB-META-ADS-MEASUREMENT (ncROAS validation)
  - Calls: KB-META-ADS-ANDROMEDA-CREATIVE-ENGINE (creative volume requirements)

**Consciousness Profile:**
- Target: Growth leads, CMOs, performance directors
- Neuro-Box: SMART (strategic scaling) + SAFE (risk-managed growth)

## Comparison: Monolithic vs KB Modules

| Dimension | Monolithic v1.0.0 | KB Modules (Proposed) |
|-----------|------------------|----------------------|
| **Total Size** | 82KB (single file) | ~375KB (7 files) |
| **Typical Load** | 7,000 tokens (L1+L2+L3) | 2,000-4,000 tokens (1-2 KBs loaded) |
| **Modularity** | Low (change one framework = reload all) | High (update one KB independently) |
| **Context Efficiency** | Low (load unused sections) | High (load only needed KBs) |
| **Parallel Execution** | No (single skill) | Yes (load Creative Engine + Image Generator simultaneously) |
| **Constitutional Compliance** | Present (in Guardrails section) | Built-in per KB (audit log per module) |
| **Consciousness Mapping** | Absent | Present (Neuro-Box 6D per KB) |
| **Dependencies** | Implicit (sister skills mentioned) | Explicit (dependency tree with integration points) |
| **Maintenance** | Hard (monolithic version bumps) | Easy (patch one KB, others unchanged) |
| **Discoverability** | Easy (one entry point) | Moderate (need orchestration or index) |
| **Learning Curve** | Low (unified mental model) | Moderate (understand 7 KB boundaries) |
