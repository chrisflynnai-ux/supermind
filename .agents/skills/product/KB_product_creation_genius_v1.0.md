## PURPOSE
This knowledge file is the **doctrine + operating system** for Product Creation Genius:
Design transformation ecosystems (90-day default), hybrid offers, and AI-wrapped methods—**validated first**, built second.

---

## L1 — QUICK REF CARD (Always Load)

### 4 Foundational Truths (Doctrine)
1) **Transformation > Information** (identity + behavior change, not content volume)
2) **Whole Solutions > Feature Lists** (Physical + Digital + Community + AI = one system)
3) **Human Core, AI Shell** (sell outcomes; AI amplifies your proprietary method)
4) **Validate First, Build Second** (72-hour profit-first validation; iterate via data)

### Non-Negotiables
- **Breakthrough in Week 1–2**
- **Support before content** (fix completion by scaffolding, not adding lessons)
- **Tier by support, not content**
- **Cross-reference everything** (no siloed components)
- **Honest timelines** (evidence-backed or framed as hypothesis/beta)

### Default Output Pack (Every time)
- Transformation Map (Breakthrough → Milestones → Miracle → Identity)
- 90-Day Architecture (Phases + week-by-week outcomes)
- Support System (4 pillars + escalation path)
- Tiered Offer (Core/Premium/VIP) + pricing bands
- Validation Plan (72 hours) + iteration metrics
- Handoffs (Email / Sales Page / Product Creation Genius / MWP)

## L2 — CORE MECHANISMS (Most Used)

### M1) Transformation Progression Map (North Star)
Breakthrough (1–2) → Milestones (3–8) → Miracle (9–12) → Transformation (identity + community)

**Rule:** If you can't state all four stages clearly, you're not ready to build curriculum.

### M2) LEARN → DO → SHARE → TEACH (Retention Amplifier)
- LEARN: 10–20 min, one big idea
- DO: same-day task, <30 min, measurable
- SHARE: community proof + accountability
- TEACH: help others / explain back → mastery + UGC

**Rule:** If completion <50%, add DO/SHARE/TEACH before adding more LEARN.

### M3) Four-Pillar Support System
Community + Coaching + Resources + AI (as friction-remover + personalization scaler)

**Rule:** If price >$200, at least **Community + Group Coaching** expected.

## L2 — CANONICAL FRAMEWORKS (10 Processes)

### F1) 90-Day Transformation Blueprint (Default Program)
- Phase 1: Foundation (Weeks 1–3)
- Phase 2: Acceleration (Weeks 4–7)
- Phase 3: Transformation (Weeks 8–12)
- Phase 4: Continuation (Alumni + next-level path)

### F2) Breakthrough → Milestone → Miracle → Transformation Map (Before Curriculum)
Define:
- Breakthrough (7–14 days)
- 2–3 milestones (Weeks 3–8)
- Miracle moment (Weeks 9–12; emotionally big, methodically achievable)
- Identity shift + belonging

### F3) Module Design: LEARN → DO → SHARE → TEACH
Every module must ship with:
- A DO task + success criteria
- A SHARE prompt
- A TEACH opportunity (reply/help/case study)

### F4) Support System Design (4 Pillars + Escalation Path)
Design:
- Community structure + rituals
- Coaching cadence + SLAs
- Resources (workbooks/templates/trackers/physical tools)
- AI roles (Q&A, reminders, progress tracking, personalization)

### F5) Whole Solution Stack (Physical + Digital + Community + AI)
Design the loop:
Physical drives behavior → Digital explains/teaches → Community normalizes → AI reduces friction → Results reinforce usage.

### F6) Tiered Hybrid Bundle Architecture (Core / Premium / VIP)
Differentiate by:
- Support level
- Personalization
- Physical depth
NOT content volume.

### F7) 3 Embedded-AI Product Types (Select Archetype)
1) AI-enhanced info product
2) AI-powered templates/tools (clear input → output)
3) Service-in-a-box (AI 80% + you 20% polish)

### F8) 72-Hour Profit-First Validation (Build Only After Signal)
- Pain capture → landing test → pre-sell waitlist
- If weak signal: refine promise/mechanism/price and retest

### F9) First 7-Day Onboarding Blueprint (Retention Window)
Day 1–7: daily touchpoints; Day 3–5 breakthrough; community activation; support clarity.

### F10) Value-Based Pricing + 3-Tier Ladder
Price anchored to outcome value, then tiered by support + depth.

## L2 — DECISION HEURISTICS (IF/THEN Rules)
1) If transformation needs 6+ months → break into stacked 90-day programs
2) If avatar is time-poor → lessons 15–20 min, tasks <30 min
3) If skeptical/burned → Week 1–2 breakthrough is non-negotiable
4) If completion <50% → support/accountability before more content
5) If adding physical → must drive critical behaviors (not "perceived value")
6) If price >$200 → include community + group coaching
7) If no waitlist/prepay → don't build; fix/pivot
8) If AI is generic → bake in proprietary method + examples
9) If first cohort → founding member, cohort launch, iterate fast
10) If tiers unclear → you differentiated on content, not support
