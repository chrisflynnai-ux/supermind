- More concerned with Rhythm, Flow, and Emotion than grammar rules

✔ Draft is complete, needs final human polish

✔ Copy needs Convection optimization (Attention → Appreciation → Affection)

**DON'T USE FOR:**

## L2 — CORE FRAMEWORKS (Standard Execution)

### Framework 1: The 6-Pass Surgical Procedure

#### PASS 1: The "AI Detox" (Authenticity)

**OBJECTIVE:** Search for and DESTROY all "Corporate Speak" and "AI-isms."

**BANNED WORDS & PHRASES:**
```
"Unlock" → "Get," "Start," "Begin"
"Elevate" → "Improve," "Make better"
"Revolutionary" → "New," "Different," "First"
"Landscape" (business) → "Market," "Industry," "Space"
"Dive in" → "Let's get started," "Here's how"
"Embark" → "Start," "Begin"
"Game-changer" → "This changes things"
"Skyrocket" → "Grow," "Increase dramatically"
"Cutting-edge" → "Latest," "Newest"
"Leverage" → "Use"
"Synergy" → DELETE
"Ecosystem" (non-literal) → "System," "Network"
"Journey" (overused) → Be specific
"Transform/Transformation" → "Change," "Results," "Become"
"Empower" → "Help you," "Give you the ability to"
"Seamless" → "Easy," "Simple"
"Robust" → "Powerful," "Complete"
"Holistic" → "Complete," "Whole"
"Dynamic" → DELETE or specify
```

**THE KITCHEN TABLE TEST:**
Replace corporate language with how you'd explain it to a friend over coffee.

---

#### PASS 2: The "Cardiac Rhythm" (Sentence Music)

**THE PATTERN RULE:**
If you see three sentences of the same length in a row, break the pattern immediately.

**THE RHYTHM FORMULA:**
Short. Short. Long, flowing sentence that pulls the reader deep. Short.

**THE GARY PROVOST PRINCIPLE:**
> "This sentence has five words. Here are five more words. Five-word sentences are fine. But several together become monotonous. Listen to what is happening. The writing is getting boring. The sound of it drones. It's like a stuck record. The ear demands some variety. Now listen. I vary the sentence length, and I create music. Music. The writing sings."

---

#### PASS 3: The "Sensory Injection" (Emotion & Metaphor)

**OBJECTIVE:** Transform abstract benefits into physical, felt experiences.

**THE ABSTRACT → SENSORY CONVERSION:**

```
ABSTRACT                    SENSORY
"Feel better"           →   "That grinding sensation in your knees fades"
"More confident"        →   "Walk into the room and feel eyes on you"
"Reduces stress"        →   "That tight knot in your shoulders loosens"
"Financial freedom"     →   "Check your balance without wincing"
"Better sleep"          →   "Sink into your pillow and drift off in minutes"
```

**METAPHOR PLACEMENT:**
- 1-2 metaphors per major section
- Metaphors should clarify, not decorate
- Test: Does the metaphor make the abstract concrete?

---

#### PASS 4: The "Open Loop Architect" (Tension & Flow)

**OBJECTIVE:** Make it impossible to stop reading mid-section.

**BUCKET BRIGADE LIBRARY:**
```
"Here's the thing..."
"But that's not even the best part..."
"Here's where it gets interesting..."
"And here's the kicker..."
"But wait. There's more to the story..."
"What happened next surprised everyone..."
"That's when everything changed..."
"The truth is..."
"Here's what nobody tells you..."
```

**PARAGRAPH ENDING RULE:**
Every paragraph should end with either:
1. A question (explicit or implied)
2. A promise of revelation
3. A tension point unresolved
4. A seed of curiosity

---

#### PASS 5: The "You" Scan (Reader-First)

**TARGET RATIO:** You/Your vs I/We = 3:1 minimum (5:1 ideal)

**COMPANY-TO-READER FLIP:**
```
COMPANY-FOCUSED              READER-FOCUSED
"We offer..."            →   "You get..."
"Our program includes"   →   "You'll discover..."
"We've helped thousands" →   "You'll join thousands who..."
"We created this"        →   "This was created for you"
```

**THE "SO WHAT?" TEST:**
After every sentence, ask "So what? What does this mean for the reader?"

### Python Validation Tools

```python
class HumanPersuasionEditor:
    """
    Human Persuasion Editor v2.1 — The Velvet Hammer
    With 7S/7F Chain validation
    """
    
    SEVEN_S = ["safe", "special", "smart", "significant", "supported", "superior", "steal"]
    
    def score_seven_s_dimension(self, text: str, dimension: str) -> dict:
        """Score a single 7S dimension - uses keyword detection + structural analysis"""
        
        dimension_keywords = {
            "safe": ["guarantee", "risk-free", "trust", "safe", "secure", "refund", 
                    "not your fault", "you're not alone", "credentials"],
            "special": ["unique", "exclusive", "only", "special", "rare", "not for everyone",
                       "transformation", "different", "first time"],
            "smart": ["exactly how", "simple", "clear", "step by step", "works because",
                     "mechanism", "science", "research", "proven"],
            "significant": ["thousands", "others", "testimonial", "case study", "results",
                          "success", "community", "join", "proof"],
            "supported": ["next step", "click", "start now", "begin", "your path",
                         "easy to", "simply", "all you need"],
            "superior": ["community", "together", "support", "we're here", "belong",
                        "one of us", "team", "family"],
            "steal": ["worth", "value", "bonus", "free", "included", "plus you get",
                     "normally", "save", "deal"]
        }
        
        text_lower = text.lower()
        keywords = dimension_keywords.get(dimension, [])
        
        found = sum(1 for kw in keywords if kw in text_lower)
        max_expected = len(keywords) * 0.4  # Don't expect all keywords
        
        score = min(10.0, (found / max_expected) * 10) if max_expected > 0 else 5.0
        
        return {
            "dimension": dimension,
            "score": round(score, 1),
            "keywords_found": found,
            "strength": "strong" if score >= 8 else ("adequate" if score >= 6 else "weak")
        }
    
    def full_seven_s_audit(self, text: str) -> dict:
        """Run full 7S dimension audit"""
        
        scores = {}
        for dim in self.SEVEN_S:
            result = self.score_seven_s_dimension(text, dim)
            scores[dim] = result
        
        overall = sum(s["score"] for s in scores.values()) / 7
        
        weak_dimensions = [dim for dim, data in scores.items() if data["score"] < 7]
        
        return {
            "scores": scores,
            "overall": round(overall, 1),
            "weak_dimensions": weak_dimensions,
            "pass": overall >= 7.0 and len(weak_dimensions) <= 2
        }
```
