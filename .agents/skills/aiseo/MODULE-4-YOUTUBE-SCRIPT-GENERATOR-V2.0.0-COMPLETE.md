## 1.1 THE 5 CORE PAINS

### Pain 1: The "Boring Hook" Death Spiral

**Symptom:** 40% drop-off in first 30 seconds. High impressions, low views.

**Root Cause:** Hook confirms expectation instead of creating EVR gap. No "Stun" factor.

**SHAQ Reframe:**
- FAQ: "How do I get more views?"
- SHAQ: "Does my hook create an Expectation vs Reality gap of 7+/10?"

---

### Pain 2: The "Invisible Expert" Syndrome

**Symptom:** Zero views from search. AI engines never cite videos.

**Root Cause:** Missing Citation Magnets (frameworks, data tables) and metadata structure (chapters, transcripts).

**SHAQ Reframe:**
- FAQ: "What tags should I use?"
- SHAQ: "Have I optimized Transcript and Chapters for AI Discovery?"

---

### Pain 3: The "AI Accent" Script

**Symptom:** Host sounds robotic. Comments say "ChatGPT script."

**Root Cause:** Using raw AI text without Velvet Hammer polish for speech. Lacks conversational rhythm.

**SHAQ Reframe:**
- FAQ: "Should I use AI to write scripts?"
- SHAQ: "Does this script pass the read-aloud Kitchen Table Test?"

---

### Pain 4: CTA Rejection

**Symptom:** High watch time, but nobody clicks. Zero conversions.

**Root Cause:** CTA feels like ad interruption, not solution. Disrupts neuro-chemical flow.

**SHAQ Reframe:**
- FAQ: "How do I sell without being salesy?"
- SHAQ: "Am I using Native Embeds aligned with PSYCHE axis?"

---

### Pain 5: The "Generalist" Trap

**Symptom:** Video goes viral to wrong audience. Wrecks conversion rate.

**Root Cause:** Optimizing for TOFU vanity metrics instead of BOFU commercial intent.

**SHAQ Reframe:**
- FAQ: "How do I go viral?"
- SHAQ: "Am I using Qualification Hooks to filter for buyers?"

## 1.3 HCTS LIBRARY

### HACKS (Fast Wins)

**H1: The "3-Hook Alignment"**
Thumbnail Visual + Video Title + First Spoken Sentence must promise exact same outcome. Misalignment = bounce.

**H2: The "Pinned Comment" Bridge**
Post comment immediately after publishing linking to Lead Magnet. Pin it. Converts 3x higher than description links.

**H3: The "Silence Breaker"**
First 5 seconds: sound effect (whoosh, pop) or visual zoom. Resets dopamine baseline, prevents scroll reflex.

**H4: The "Chapter SEO"**
Name chapters as questions ("What are BOFU keywords?"). AI indexes specific moments.

### CRACKS (Failure Points)

**C1: The "Welcome Back" Intro**
"Welcome back to the channel, guys!" = retention killer. New viewers don't care. Start with Hook at 0:00.

**C2: The "Desktop Recording" Snooze**
Static screen >10 seconds. Use Pattern Interrupts (zoom, highlight, face cam) every 5-10 seconds.

**C3: The "Premature Goodbye"**
"In summary..." or "Thanks for watching" = viewers leave before CTA. Use "Fortune Cookie" transition instead.

**C4: The "AI Accent" Script**
"Unlock the potential," "leverage your assets" = robotic. Replace with spoken bridges.

### TRACKS (Paths to Authority)

**T1: The "WSA Repurpose" Workflow**
Module 3 Article → Extract 7 Sections → Map to Video Chapters → Record. Deep content ready to film.

**T2: The "Series" Strategy**
Don't make one video. Make 3-part series (Problem, Mechanism, Case Study). Builds authority + binge sessions.

**T3: The "Named Framework" Play**
Create citeable framework shown on screen. AI OCR reads frames. Becomes intellectual property.

### STACKS (Tool Combos)

**S1: Viral Script Stack**
Module 2 Keywords + Module 3 WSA + KB-VIRAL-HOOK (templates)

**S2: Optimization Stack**
YouTube Studio (chapters) + Claude (transcript cleanup) + Schema Validator (JSON-LD)

**S3: Distribution Stack**
Long-form → Opus Clip → 5 Short-form clips → Multi-platform

## 2.3 PHASE 1: PRE-PRODUCTION FOUNDATION
**Day 1 Morning — 3 hours**

### Step 1.1: Load Inputs from Module 3 (30 min)

- Import WSA Article
- Extract 7 core sections
- Identify 3 most surprising insights as "EVR Candidates"
- Pull BOFU keyword data from Module 2

### Step 1.2: EVR Psychology Planning (45 min)

```yaml
evr_planning:
  current_expectation: "What viewer believes about topic"
  reality_contrast: "Surprising truth with data"
  evr_score: "[0-10, target 7+]"
  
  quality_gate: "If score <7, iterate on reality contrast"
```

### Step 1.3: Story Archetype Selection (30 min)

**12 Archetypes:**
1. Breakdown (failure → lesson)
2. Case Study (client results)
3. Problem → Solution
4. Tutorial (step-by-step)
5. Listicle (5 best X)
6. Comparison (X vs Y)
7. Controversy (unpopular opinion)
8. Behind-the-scenes
9. Reaction (respond to trend)
10. Interview (expert insights)
11. Transformation (before → after)
12. Data storytelling (research)

**Match to Content + Avatar Stage:**
- Comparing solutions → Comparison
- Knows problem, not fix → Tutorial
- Believes conventional wisdom → Contrarian

### Step 1.4: Neuro-Axis Video Mapping (45 min)

| Timestamp | Axis | Goal | Tactic |
|-----------|------|------|--------|
| 0:00-1:00 | BODY | Safety | "Not selling you" |
| 1:00-3:00 | HEART | Emotion | Story of failure |
| 3:00-8:00 | MIND | Logic | Framework |
| 8:00-10:00 | SPIRIT | Proof | Case studies |
| 10:00-12:00 | PSYCHE | Action | Next steps |
| 12:00-end | CONSCIENCE | Belonging | Community |

**Balance Check:** Excitatory vs Inhibitory difference ≤4 points

### Step 1.5: Format Decision (30 min)

- Video Length (archetype-based)
- Visual Style (face cam, screen share, B-roll)
- Production Tier (Simple, Standard, Advanced)

## 2.4 PHASE 2: HOOK ENGINEERING
**Day 1 Afternoon — 2.5 hours**

### Step 2.1: Generate 20 Hook Variations (60 min)

**Use 20 Hook Templates:**

1. **Unpopular Opinion:** "Unpopular opinion: [Practice] is destroying your [Result]."
2. **Time Donation:** "If you're still doing [Old], you're donating money to competitors."
3. **Authority Secret:** "I asked 50 [Experts] about [Topic], and they all said..."
4. **I Spent:** "I spent $50k on [Tool] so you don't have to."
5. **Belief Reversal:** "I used to believe X. Now I believe Y."
6. **The Warning:** "Stop doing [X] before it's too late."
7. **Data Shock:** "[Number]% of [Group] are doing [X] wrong."
8. **Confession:** "I made $X doing something everyone says not to."
9. **Myth Buster:** "Everything you know about [Topic] is wrong."
10. **Time Machine:** "What I wish I knew [X years] ago about [Topic]."
11. **Insider:** "The [Industry] secret they don't want you to know."
12. **Comparison Bait:** "[Popular Thing] vs [Your Thing]: The truth."
13. **Challenge:** "Try this for 7 days and watch what happens."
14. **Question:** "Why do [Experts] fail at [Thing]?"
15. **Story Hook:** "I lost everything, then discovered [X]."
16. **List Tease:** "5 [Things] that will [Outcome] (most people miss #3)."
17. **Negative:** "[Thing] is dead. Here's what works now."
18. **Authority Challenge:** "[Big Name] is wrong about [Topic]. Here's why."
19. **Specific Result:** "How I [Achieved X] in [Time] without [Common Method]."
20. **FOMO:** "[Number] people already know this. Do you?"

**Add Quantification:** Every hook needs a number.

### Step 2.2: Hook Quality Audit (30 min)

**8-Point Checklist:**
- [ ] Single Subject?
- [ ] Single Question?
- [ ] 3-Hook Alignment?
- [ ] EVR Gap Score 7+?
- [ ] Stun Type Present?
- [ ] Neuro-Axis Identified?
- [ ] BOFU Relevant?
- [ ] No AI-Accent Words?

Select Top 3 Hooks (Primary + 2 Variants)

### Step 2.3: Thumbnail Storyboarding (60 min)

**4 Stun Types:**
1. **Person Stun:** Recognizable face with emotion
2. **Recognizable Stun:** Famous brand/object
3. **Atypical Visual:** Unexpected imagery
4. **Atypical Layout:** Breaking composition rules

**Formula:** [RECOGNIZABLE] + [EMOTION/DATA] + [TEXT OVERLAY]

**Verify 3-Hook Alignment:** Visual + Title + First 15s = Same promise

## 2.6 PHASE 4: SCRIPT WRITING
**Day 2 Afternoon — 4 hours**

### Step 4.1: Write BODY First (2 hours)

**Principle:** Can't write compelling hook until you know what you're hooking into.

- Expand chapter outlines into full script
- Write conversationally
- Add B-Roll notes: Change visual every 5-10 seconds
- Insert Data Tables and Citation Magnets (show on screen)

### Step 4.2: Write INTRO Second (60 min)

**EVR Formula:**
```
Pattern Interrupt (0:00-0:05)
+ EVR Statement (0:05-0:15)
+ Credibility Anchor (0:15-0:25)
+ Promise (0:25-0:35)
+ Curiosity Gap (0:35-0:45)
+ Thesis Statement (0:45-1:00)
```

Verify 3-Hook Alignment

### Step 4.3: Write OUTRO Last (45 min)

**Fortune Cookie Insight:** Zoom out to meta-lesson

**Native Embed CTA:**
```
[Identify natural next step]
+ [Position resource as solution to pain raised]
+ [Clear instruction]
```

**Loop to Next Video:** "If you want to learn X next, watch this..."

### Step 4.4: Add Visual Cues (15 min)

**Pattern Interrupt Schedule:** Every 45-60 seconds
- Sound effect
- Zoom
- Color flash
- Text overlay
- Face cam cut

1. Stand up

1. Record yourself reading

**Symptom:** Invisible to non-subscribers, no AI citations

**Timeline:** 3.5 hours

**Symptom:** High AVD, zero clicks
**Recovery:** Rewrite as Native Embed, move to 80-90% mark, add Fortune Cookie
**Timeline:** 1.5 hours

### FM6: Viral-But-Wrong-Audience
**Symptom:** High views, zero conversions
**Recovery:** Add qualification filter in first 60s, sacrifice virality for BOFU intent
**Timeline:** 2 hours
