Failure Mode 1: The "Vanity Volume" Trap

Symptom:
  • Targeting keywords with 50k+ searches/month
  • Low conversion rate (<0.5%)
  • High bounce rate (>70%)
  • Stakeholder says: "We're getting traffic but no leads"

Root Cause:
  • Traditional SEO mindset prioritizing search volume over commercial intent
  • Targeting informational queries that AI Mode answers instantly
  • Confusing "awareness" with "demand"

Diagnostic Questions:
  1. What percentage of target keywords contain commercial modifiers (best/vs/pricing)?
     - If <20%: You're in the Vanity Trap
  2. What is the average position of keywords with conversions?
     - If converting keywords rank better than non-converting: You're prioritizing wrong keywords
  3. Do your top traffic pages have clear CTAs?
     - If no: Content is informational, not commercial

Real-World Example:
  • B2B SaaS company targeting "SEO" (100k searches/mo)
  • Ranked #8, got 2,000 visits/month, 0 demos
  • Pivoted to "SaaS SEO agency pricing" (200 searches/mo)
  • Ranked #3, got 50 visits/month, 5 demos
  • Lesson: 200 targeted visits > 2,000 waste visits

Step-by-Step Fix (Recovery Protocol):
  1. Export all keywords from GSC (last 90 days)
  2. Apply p_bofu_regex_filter to isolate commercial intent
  3. Delete any keyword scoring <4/10 on the Wallet Proximity scale
  4. Audit existing content:
     - Which pages target informational queries? (Archive or no-index these)
     - Which pages target BOFU queries? (Expand these to WSA format)
  5. Redirect resources:
     - Stop creating "What is X" content
     - Start creating "Best X for Y" and "X vs Y" comparisons
  6. Track new metrics:
     - Conversion rate per keyword (target: >2%)
     - Average deal value per keyword source
     - MRR from organic (not just traffic volume)

Prevention:
  • Use the MMA Framework from the start (Money, Movement, Authority)
  • Never approve a keyword list where <50% have commercial modifiers
  • Set stakeholder expectation: "We're optimizing for revenue, not traffic"

Tools:

  • Wallet Proximity Scoring Rubric (KB-BOFU-TARGETING-SYSTEM)
  • Conversion tracking in GA4 (set up goal for "Demo Request")

Timeline to Recovery:

  • Month 2-3: Start seeing conversion rate improvements
  • Month 4-6: Traffic may drop, but revenue increases
