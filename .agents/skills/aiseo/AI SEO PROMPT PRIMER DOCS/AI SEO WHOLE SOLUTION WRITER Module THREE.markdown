3. Knowledge Blocks (Deep Dive Frameworks)
KB-WSA-ARCHITECTURE-v1.0 (5-6 Pages)
1. The WSA Philosophy: Authority Over Volume
Ģ Concept: A Whole Solution Article (WSA) is not a blog post. It is a comprehensive asset designed to be the definitive answer to a specific problem.
Ģ The "Answer Every Question" Principle: A WSA must anticipate the user's next 3 questions and answer them immediately (Query Fan-out).
Ģ Depth vs. Length: Depth is not word count; it is "Information Gain." Does this article provide a new angle, data point, or framework that doesn't exist elsewhere?
2. The 7-Section Architecture
Ģ Section 1: Safety Anchor (BODY/Dopamine): Establish credibility and safety immediately. "I'm not here to sell you... I've tested this myself."
Ģ Section 2: Transformation Hook (HEART/GABA): The emotional "After" state. "Imagine if you never had to worry about [Pain] again."
Ģ Section 3: Mechanism Explained (MIND/Acetylcholine): The logic. "Here is exactly how this works (The Framework)."
Ģ Section 4: Proof Layer (SPIRIT/Serotonin): Social validation. "Here are 3 others who achieved this."
Ģ Section 5: Action Path (PSYCHE/Adrenaline): The step-by-step protocol. Frictionless action.
Ģ Section 6: Community Promise (CONSCIENCE/Oxytocin): "You are not alone. Join the tribe."
Ģ Section 7: Value Stack (The "Steal"): The irresistible offer. "Get the tool + the course + the template."
3. BOFU Content Types
Ģ Comparison Pages: "X vs Y." Structure: Direct feature matrix + unique "Winner" verdict based on use case.
Ģ "Best Of" Collections: "Best X for [Persona]." Structure: Criteria for selection + Data Table + Individual Reviews.
Ģ Problem-Solution: "How to Fix X." Structure: Diagnosis -> Root Cause -> Protocol -> Prevention.
4. Schema & AI Citation Optimization
Ģ FAQ Schema: Use JSON-LD to markup questions/answers. AI agents prioritize this for "Direct Answers."
Ģ Tables: HTML tables are the #1 most cited element in AI Overviews. Always use them for pricing, features, or pros/cons.
Ģ Entity Definitions: Use Semantic Triples (Subject-Predicate-Object) in the first paragraph.

KB-NEURO-RESONANCE-WRITING-v1.0 (5-6 Pages)
1. The Corrected Neuro-Box 6D Model
Ģ BOTTOM (BODY/Dopamine): Safety, Status, Resource Acquisition. Question: "Am I safe? Will this elevate me?"
Ģ TOP (HEART/GABA): Emotion, Connection, Calm. Question: "Do I feel this? Is it unique?"
Ģ LEFT (MIND/Acetylcholine): Logic, Focus, Validation. Question: "Does this make sense? Is it smart?"
Ģ RIGHT (SPIRIT/Serotonin): Significance, Dominance, Proof. Question: "Do others respect this? Is it proven?"
Ģ FRONT (PSYCHE/Adrenaline): Action, Urgency, Fight/Flight. Question: "Can I act now?"
Ģ BACK (CONSCIENCE/Oxytocin): Tribe, Support, Obligation. Question: "Do I belong? Am I supported?"
2. The 5 Purchase Driver Sequence
Ģ Sequence Rule: You cannot skip steps.
    1. Higher Status (Dopamine): "This makes you better." (Intro)
    2. Emotional Stimulation (GABA): "This feels good." (Hook)
    3. Stress Relief (Acetylcholine): "This solves the confusion." (Mechanism)
    4. Social Validation (Serotonin): "Others are winning." (Proof)
    5. The "Steal" (All Axes): "The value outweighs the cost." (Offer)
3. The 7S Chain Validation
Ģ SAFE: Guarantee, credentials, transparency.
Ģ SPECIAL: Unique mechanism, emotional resonance.
Ģ SMART: Clarity, logic, data.
Ģ SIGNIFICANT: Testimonials, social proof.
Ģ SUPPORTED: Friction removal, clear path.
Ģ SUPERIOR: Belonging, elite status.
Ģ STEAL: Value stack, price anchoring.
4. Balance Checking
Ģ Excitatory vs. Inhibitory: You must balance "Hype" (Excitatory) with "Calm/Safety" (Inhibitory).
Ģ Example: High Adrenaline (Urgency) without Oxytocin (Support) feels like a scam. High Dopamine (Promise) without Acetylcholine (Logic) feels like vaporware.
