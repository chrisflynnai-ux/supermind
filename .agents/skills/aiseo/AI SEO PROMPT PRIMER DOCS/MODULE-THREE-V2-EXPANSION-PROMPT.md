CONVECTION vs CONVERSION: The Paradigm Shift

├── Goal: Create heat that rises naturally (earn the right to sell)
├── Tactics: Attention→Appreciation→Affection (earned at each stage)
├── Psychology: Pull them toward you by demonstrating value first
├── Result: Zero buyer's remorse, advocacy, repeat customers, referrals
