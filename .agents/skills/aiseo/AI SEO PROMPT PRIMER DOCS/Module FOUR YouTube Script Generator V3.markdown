1. The 5 Core Pains (Video Strategy Deep Dive)
• Pain 1: The "Boring Hook" Death Spiral
    ? Symptom: "We have great information, but 40% of viewers drop off in the first 30 seconds."
    ? Root Cause: The hook confirms the expectation but fails to create a "Reality Gap" (EVR). It lacks the "Stun" factor required to stop the scroll.
    ? Strategic Implication: You must engineer the first 60 seconds using 3-Hook Alignment (Visual + Spoken + Text) and EVR Psychology to buy the viewer's attention for the rest of the video.
• Pain 2: The "Invisible Expert" Syndrome
    ? Symptom: "We publish videos, but they get zero views from search and AI engines never cite them."
    ? Root Cause: Missing "Citation Magnets" (Frameworks, Data Tables) and metadata structure (Chapters, Transcripts) that AI agents use to index video content.
    ? Strategic Implication: Video scripts must be written for AI readability (clear structure, named frameworks) as much as human entertainment.
• Pain 3: The "AI Accent" Script
    ? Symptom: "Our host sounds robotic. Comments say it feels like a ChatGPT script."
    ? Root Cause: Using raw AI text for scripts without applying the Velvet Hammer 6-Pass Polish adapted for speech. It lacks sensory language and "Kitchen Table" conversational rhythm.
    ? Strategic Implication: Scripts must pass a "read-aloud" test where "written" words (e.g., "furthermore," "optimize") are replaced with "spoken" bridges (e.g., "and here's the thing").
• Pain 4: CTA Rejection
    ? Symptom: "High watch time, but nobody clicks the link or buys the product."
    ? Root Cause: The Call to Action (CTA) feels like an interruption (Ad) rather than a solution (Native Embed). It disrupts the neuro-chemical flow.
    ? Strategic Implication: Use Native Embeds where the free resource/tool is positioned as the natural next step to solve the pain point raised in the video.
• Pain 5: The "Generalist" Trap
    ? Symptom: "A video goes viral to the wrong audience, wrecking our conversion rate."
    ? Root Cause: Optimizing for TOFU (Top of Funnel) vanity metrics instead of BOFU (Bottom of Funnel) commercial intent.
    ? Strategic Implication: Sacrifice broad virality for Targeted Resonance. Use "Qualification Hooks" ("This is specifically for SaaS founders...") to filter for the right buyer.

F. Failure Modes (Diagnostics & Recovery)
Failure Mode 1: The "Boring Hook" Death Spiral
• Symptom: CTR <3% or 40% drop-off in first 30 seconds. Impressions are high but views are low.
• Root Cause: Hook confirms expectation instead of creating an EVR gap.
• Diagnostic Questions: What is the EVR score? Is it <7? Do the 3 hooks align?
• Recovery:
    1. Calculate current EVR score.
    2. Find the hidden EVR (most surprising fact buried in video).
    3. Rewrite hook using 20 Templates.
    4. Create new thumbnail featuring data point.
    5. A/B test new title/thumbnail.
    6. Add pattern interrupt at 0:05.
• Timeline: 4 hours active work.
Failure Mode 2: Retention Cliff
• Symptom: Sharp drop (>20%) at a specific timestamp.
• Root Cause: Abrupt topic transition without grease-slide connector, or "goodbye" signal sent too early.
• Diagnostic Questions: Is there a connector at the drop point? Is there a visual change?
• Recovery:
    1. Identify exact cliff timestamp.
    2. Diagnose cause (visual static? tone shift?).
    3. Add grease-slide connector ("But here's the part that surprised me...").
    4. Add pattern interrupt (sound/zoom).
    5. Remove premature goodbye signals.
    6. Re-upload or add pinned comment addressing the section.
• Timeline: 2 hours.
Failure Mode 3: "AI-Accent" Video
• Symptom: Comments say "robotic" or "ChatGPT script." High bounce rate despite good hook.
• Root Cause: Script written as text first, used AI words (utilize, leverage), not read aloud.
• Diagnostic Questions: Are there banned words? Is sentence length >18 words?
• Recovery:
    1. Run AI Word Detector.
    2. Apply Velvet Hammer 6-Pass.
    3. Add natural pauses.
    4. Refilm in "conversation mode" (not reading mode).
    5. Add vocal inflection in edit.
• Timeline: 5-6 hours (includes re-filming).
Failure Mode 4: Zero Search/AI Discovery
• Symptom: Video invisible to non-subscribers; no AI citations. Google doesn't index chapters.
• Root Cause: Title/description not optimized, no FAQ schema, chapters not questions.
• Diagnostic Questions: Is primary keyword in title? Are chapters questions? Is schema present?
• Recovery:
    1. Audit keyword usage.
    2. Optimize title (front-load keyword).
    3. Rewrite chapters as questions.
    4. Cleanup transcript filler words.
    5. Create blog post with embedded video + schema.
    6. Add citation magnets (named frameworks).
• Timeline: 3.5 hours.
Failure Mode 5: CTA Rejection
• Symptom: High AVD but zero clicks. End screen CTR <2%.
• Root Cause: CTA feels like an ad break. Broken neuro-chemical flow.
• Diagnostic Questions: Is CTA positioned as a solution? Is it a native embed?
• Recovery:
    1. Identify CTA placement.
    2. Rewrite as Native Embed (Agitate pain -> Position resource as solve).
    3. Move CTA to 80-90% mark.
    4. Add "Fortune Cookie" transition.
    5. Test new CTA (pinned comment or new video).
• Timeline: 1.5 hours.
Failure Mode 6: Viral-But-Wrong-Audience
• Symptom: High views, zero conversions. Comments from general public, not ICP.
• Root Cause: Optimized for TOFU virality instead of BOFU intent. No qualification filter.
• Diagnostic Questions: Did hook qualify the audience? Is topic BOFU?
• Recovery:
    1. Analyze audience demographics.
    2. Identify qualification gap.
    3. Create BOFU viral hook (Contrarian + Filter + Outcome).
    4. Add disqualification statement in first 60s.
    5. Sacrifice virality for conversion on next video.
• Timeline: 2 hours to pivot strategy.
