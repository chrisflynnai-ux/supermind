**Team:** 1 strategist/writer (can be same person)

### Phase 1: Pre-Production Foundation (Day 1 Morning - 3 hours)

**Goal:** Establish psychological foundation and structural framework before writing a single word.

#### Step 1.1: Load Inputs from Module 3 (30 minutes)

**Action 1:** Import WSA Article
- Open the Module 3 WSA article
- Extract the 7 core sections:
  1. Safety Anchor
  2. Mechanism Reveal
  3. Proof Stack
  4. Action Path
  5. Objection Kill
  6. Value Stack
  7. Closer
- Copy into new document titled "Video Source Material"

**Action 2:** Identify Core Insights
- Read through WSA
- Highlight the 3 most surprising/valuable insights
- Ask: "What made me go 'Whoa, I didn't know that' when researching this?"
- Mark these as "EVR Candidates"

**Action 3:** Extract BOFU Keyword Data
- Pull primary keyword from Module 2
- Note: Search volume, KD, current ranking
- Identify semantic variations (e.g., "best SEO tools" → "top SEO software," "SEO tool comparison")

**Tool:** Google Docs or Notion for source material aggregation

**Output:** Single document with WSA content + 3 EVR candidates + keyword data

**Quality Gate:** Can you explain the article's value in one sentence? If not, re-read.

#### Step 1.2: EVR Psychology Planning (45 minutes)

**Action 1:** Define Current Expectation (15 min)
- What does your target avatar currently believe about this topic?
- What is the "safe" conventional wisdom they've been taught?
- What would they expect a video on this topic to say?

**Worksheet:**
```
Target Avatar: [SaaS Founder, 0-$50K MRR, trying to grow with SEO]

Current Expectation:
"I need to target high-volume keywords to get lots of traffic. More traffic = more trials."

Source of Expectation:
- Generic SEO blogs
- Tool dashboards that highlight volume
- "Get to #1 on Google" mentality
```

**Action 2:** Craft the Reality Contrast (20 min)
- What data/experience contradicts this expectation?
- What is the surprising truth?
- Can you quantify the gap?

**Worksheet:**
```
Reality:
"High-volume keywords convert at 0.3% for SaaS trials. Low-volume BOFU keywords convert at 8.7%. Targeting volume is costing you $37K in MRR."

Data Source:
- Analysis of 247 SaaS sites (cite real study or your own data)
- Your own case study (127 trials from low-volume keywords)
```

**Action 3:** Score the EVR Gap (10 min)
- Use the 0-10 scale
- Test on 2-3 people in target audience
- Ask: "On scale of 1-10, how surprising is this claim?"
- **Target:** 7+

**Worksheet:**
```
EVR Score: 8/10
Reasoning: Most SaaS founders are told the opposite. This challenges identity ("Am I wasting time?") and offers quantified cost.
```

**Tool:** Spreadsheet or Notion database to track EVR scores

**Output:** Completed EVR Worksheet with 7+ score

**Quality Gate:** If score <7, iterate on the reality contrast. Add more specificity or data.

#### Step 1.3: Story Archetype Selection (30 minutes)

**Action 1:** Review the 12 Archetypes
- Comparison (X vs Y)
- Tutorial (How to do X)
- Contrarian (Why X is wrong)
- Case Study (How I/we did X)
- Listicle (Top X tools)
- Problem → Solution
- Alternative/Migration
- Deep Dive (Ultimate Guide)
- Behind-the-Scenes
- Mistake Breakdown
- Ranking/Rating
- Scenario (If I had to start over)

**Action 2:** Match Archetype to Content + Avatar Stage

**Decision Tree:**
```
IF avatar is comparing solutions → Use Comparison
IF avatar knows the problem but not the fix → Use Tutorial or Problem → Solution
IF avatar believes conventional wisdom → Use Contrarian
IF avatar needs proof → Use Case Study
IF avatar is researching options → Use Listicle or Ranking
IF avatar wants comprehensive knowledge → Use Deep Dive
IF avatar is dissatisfied with current solution → Use Alternative/Migration
```

**Action 3:** Select Primary and Backup Archetype

**Example Decision:**
```
Primary: Tutorial (How to Find BOFU Keywords)
Reasoning: Avatar knows they need keywords but doesn't know the process

Backup: Contrarian (Why High-Volume Keywords Are Killing Your Growth)
Reasoning: If tutorial angle feels too generic, pivot to contrarian to create EVR
```

**Tool:** Decision flowchart (draw on paper or use Miro/FigJam)

**Output:** Selected archetype with reasoning documented

**Quality Gate:** Does this archetype naturally support your EVR gap? If not, reconsider.

#### Step 1.4: Neuro-Axis Video Mapping (45 minutes)

**Action 1:** Create Timeline Grid (15 min)

**Template (for 10-minute video):**
```
| Minute | Neuro-Axis | Content Type | Goal |
|--------|-----------|--------------|------|
| 0-1    | BODY      | Hook + Safety | "This is for you, it's safe" |
| 1-2    | HEART     | Pain Agitation | "Here's why it matters emotionally" |
| 2-4    | MIND      | Framework/Logic | "Here's how it works" |
| 4-6    | SPIRIT    | Proof/Social | "Here's who else succeeded" |
| 6-8    | PSYCHE    | Action Steps | "Here's exactly what to do" |
| 8-9    | CONNECTION| Identity/Tribe | "You're part of the winners now" |
| 9-10   | SPIRIT    | Transcendence | "Here's the bigger lesson" |
```

**Action 2:** Map WSA Sections to Timeline (15 min)

**Example Mapping:**
```
WSA Safety Anchor → Video Minute 0-1 (BODY axis)
WSA Mechanism → Video Minute 2-4 (MIND axis)
WSA Proof Stack → Video Minute 4-6 (SPIRIT axis)
WSA Action Path → Video Minute 6-8 (PSYCHE axis)
```

**Action 3:** Check Excitatory/Inhibitory Balance (15 min)

**Count:**
- Excitatory: HEART (1) + PSYCHE (2) + CONNECTION (1) = 4
- Inhibitory: BODY (1) + MIND (2) + SPIRIT (2) = 5
- Difference: 5 - 4 = 1 point ✅ (Target: ≤4)

**Adjustment Rules:**
- If difference >4: Add more logic (MIND) or safety (BODY) if too excitatory
- If difference <-4: Add more emotion (HEART) or action (PSYCHE) if too inhibitory

**Tool:** Excel or Google Sheets neuro-axis timeline template

**Output:** Completed timeline map with balance check

**Quality Gate:** Balance difference ≤4 points? All 6 axes represented at least once?

#### Step 1.5: Format Decision (30 minutes)

**Action 1:** Determine Video Length

**Formula:**
```
IF Archetype = Comparison → 8-12 minutes
IF Archetype = Tutorial → 5-10 minutes
IF Archetype = Case Study → 8-15 minutes
IF Archetype = Deep Dive → 15-25 minutes
IF Archetype = Listicle → 7-12 minutes
```

**Complexity Modifier:**
```
IF topic requires >5 steps → Add 2-3 minutes
IF topic has objections to address → Add 1-2 minutes
IF topic needs social proof → Add 1-2 minutes
```

**Example:**
```
Base (Tutorial): 5-10 minutes
+ 5 steps to show: +2 minutes
+ Objection handling: +1 minute
+ Case study proof: +2 minutes
= Target: 10-15 minutes
```

**Action 2:** Determine Visual Style

**Decision Matrix:**
```
IF B2B high-ticket (>$5K deal) → Face cam + screen share (builds trust)
IF Tutorial/How-to → Screen recording with voice-over
IF Contrarian/Thought leadership → Face cam primary, B-roll secondary
IF Comparison → Split screen or side-by-side visuals
```

**Action 3:** Decide on Production Complexity

**Tiers:**
```
Tier 1 (Simple): Webcam or screen recording, basic editing
Tier 2 (Standard): DSLR + lapel mic, B-roll, on-screen text
Tier 3 (Advanced): Multi-cam, motion graphics, professional editing

Recommendation: Start Tier 1-2 for first 10 videos, upgrade based on performance data
```

**Tool:** Production planning doc

**Output:** Format spec (length, visual style, production tier)

**Quality Gate:** Is this sustainable? Can you produce 1-2 videos/month at this quality?

### Phase 3: Script Outline (Day 2 Morning - 2 hours)

**Goal:** Build skeletal structure with chapters, connectors, and uniqueness validation.

#### Step 3.1: Uniqueness Gate Check (30 minutes)

**The Principle:** If your outline isn't novel, don't write the script—go back and research more.

**Action 1:** Search Competitor Videos (15 min)
- Go to YouTube
- Search your primary keyword
- Watch top 3 videos (on 2x speed)
- Note their outline/structure

**Competitor Analysis Template:**
```
Video 1: "Best SEO Tools for SaaS"
- Structure: 5-tool list, feature rundown, pricing
- Unique angle: None (generic list)
- Weakness: No use-case matching, no specific avatar

Video 2: "SEO Tools Comparison 2026"
- Structure: Side-by-side comparison table
- Unique angle: Direct comparison (better than list)
- Weakness: No implementation guidance

Video 3: "I Tested 8 SEO Tools"
- Structure: Testing process, results reveal
- Unique angle: First-hand testing (strong)
- Weakness: No SaaS-specific focus
```

**Action 2:** Find Your Uniqueness Angle (15 min)

**Uniqueness Types:**
1. **New Information:** Data/insight competitors don't have
2. **New Delivery:** Same info, better structure (e.g., framework vs. list)
3. **New Phrasing:** Reframe the concept (e.g., "BOFU targeting" vs. "keyword research")
4. **New Examples:** Specific use cases competitors miss

**Your Unique Angle:**
```
Angle: SaaS-Specific BOFU Keyword Process (5 steps)
Why Unique:
- Competitors show tools, not process
- Competitors target general SEO, not SaaS specifically
- I have data from 247 SaaS sites they don't have
- I have proprietary BOFU filter they can't replicate
```

**Gate Decision:**
- ✅ PASS: Unique angle identified → Proceed to outline
- ❌ FAIL: Too similar to competitors → Go back to research, find new data/angle

**Tool:** Competitor analysis spreadsheet

**Output:** Documented unique angle that passes gate

### Phase 4: Script Writing (Day 2 Afternoon - 4 hours)

**Goal:** Write full script in correct order (Body → Intro → Outro), add visual cues.

#### Step 4.1: Write Body FIRST (2 hours)

**The Principle:** You can't write a compelling hook until you know what you're hooking people into.

**Action 1:** Expand Chapter Outlines into Full Script (90 min)

**Process:**
- Start with Chapter 3 (First major point)
- Write conversationally as if explaining to a friend
- Include specific examples, data, screenshots
- Add visual cues in [brackets]

**Example Script Segment:**

```
CHAPTER 3: Step 1 - Export Your GSC Data (2:00-3:30)

[VISUAL: Screen recording of Google Search Console]

Alright, Step 1 is dead simple but most people skip it: Export all your keywords from Google Search Console.

[VISUAL: Click Performance tab]

Go to Performance, click on the Queries tab, and you'll see every single keyword that's driven at least one impression to your site.

[ON-SCREEN TEXT: "Queries Tab"]

Now here's the thing most people don't realize: 70-80% of these keywords have ZERO commercial intent. They're people asking "what is SaaS" or "how does SEO work." Those are TOFU keywords. They drive traffic but zero trials.

[VISUAL: Highlight low-intent keywords with red X]

We need to filter these out. But before we do, export the full list.

[VISUAL: Click Export button]

Click Export, download as CSV, and open it in Google Sheets.

[VISUAL: Google Sheets with data]

You should see columns for Query, Clicks, Impressions, CTR, and Position.

[GREASE-SLIDE: "Now this is where it gets interesting..."]

Most keyword research starts with guessing what people search for. But you already KNOW what people are searching for—because they're finding you. We just need to find the high-intent needles in this haystack.

[TRANSITION to Chapter 4]
```

**Writing Rules:**
- Speak in second person ("you") not first person ("I")
- Use contractions (it's, you're, don't) for conversational tone
- Keep sentences <20 words for easy speaking
- Avoid written-only words (e.g., "utilize" → "use", "obtain" → "get")

**Action 2:** Add B-Roll and Visual Notes (20 min)

**Visual Cue Types:**
```
[VISUAL: Description] - What to show on screen
[ON-SCREEN TEXT: "Text"] - Text overlay to display
[B-ROLL: Description] - Supplemental footage
[ZOOM: Element] - Camera direction
[CUT TO: Scene] - Scene change
```

**Rule:** Change visual every 5-10 seconds

**Action 3:** Insert Data Tables and Citation Magnets (10 min)

**Example:**
```
[DATA TABLE ON SCREEN - Display for 8 seconds:
TOFU Keywords: 0.3% conversion, 50K volume, $0.12 CPA
BOFU Keywords: 8.7% conversion, 200 volume, $37 CPA]

As you can see in this data from 247 SaaS sites, BOFU keywords convert at 8.7% compared to just 0.3% for TOFU keywords. That's a 29x difference.
```

**Tool:** Google Docs with formatting for visual cues

**Output:** Full body script (Points 1-3) with visual cues

**Quality Gate:** Read aloud. Does it sound natural? Can you speak it without stumbling?

**Action 2:** Write Native Embed CTA (15 min)

```
[Identify the natural next step] + [Position resource as solution] + [Clear instruction]
```

```

Now, the exact BOFU regex filter I showed you in Step 2? I built a free Google Sheets template that does it automatically.
```
