1. The 5 Core Pains (Video Strategy Deep Dive)
• Pain 1: The "Boring Hook" Death Spiral
    ? Symptom: "We have great information, but 40% of viewers drop off in the first 30 seconds."
    ? Root Cause: The hook confirms the expectation but fails to create a "Reality Gap" (EVR). It lacks the "Stun" factor required to stop the scroll.
    ? Strategic Implication: You must engineer the first 60 seconds using 3-Hook Alignment (Visual + Spoken + Text) and EVR Psychology to buy the viewer's attention for the rest of the video.
• Pain 2: The "Invisible Expert" Syndrome
    ? Symptom: "We publish videos, but they get zero views from search and AI engines never cite them."
    ? Root Cause: Missing "Citation Magnets" (Frameworks, Data Tables) and metadata structure (Chapters, Transcripts) that AI agents use to index video content.
    ? Strategic Implication: Video scripts must be written for AI readability (clear structure, named frameworks) as much as human entertainment.
• Pain 3: The "AI Accent" Script
    ? Symptom: "Our host sounds robotic. Comments say it feels like a ChatGPT script."
    ? Root Cause: Using raw AI text for scripts without applying the Velvet Hammer 6-Pass Polish adapted for speech. It lacks sensory language and "Kitchen Table" conversational rhythm.
    ? Strategic Implication: Scripts must pass a "read-aloud" test where "written" words (e.g., "furthermore," "optimize") are replaced with "spoken" bridges (e.g., "and here's the thing").
• Pain 4: CTA Rejection
    ? Symptom: "High watch time, but nobody clicks the link or buys the product."
    ? Root Cause: The Call to Action (CTA) feels like an interruption (Ad) rather than a solution (Native Embed). It disrupts the neuro-chemical flow.
    ? Strategic Implication: Use Native Embeds where the free resource/tool is positioned as the natural next step to solve the pain point raised in the video.
• Pain 5: The "Generalist" Trap
    ? Symptom: "A video goes viral to the wrong audience, wrecking our conversion rate."
    ? Root Cause: Optimizing for TOFU (Top of Funnel) vanity metrics instead of BOFU (Bottom of Funnel) commercial intent.
    ? Strategic Implication: Sacrifice broad virality for Targeted Resonance. Use "Qualification Hooks" ("This is specifically for SaaS founders...") to filter for the right buyer.

C. HCTS Library (Video Tactics)
HACKS (Fast Wins)
1. The "3-Hook Alignment": Ensure your Thumbnail Visual, Video Title, and First Spoken Sentence all promise the exact same outcome. Misalignment causes immediate bounce.
2. The "Pinned Comment" Bridge: Post a comment immediately after publishing that links to your Lead Magnet (Module 5) and pin it. This converts 3x higher than description links.
3. The "Silence Breaker": In the first 5 seconds, use a sound effect (whoosh, pop) or visual zoom. This resets the viewer's dopamine baseline and prevents "scroll reflex".
CRACKS (Failure Points)
1. The "Welcome Back" Intro: Starting a video with "Welcome back to the channel, guys!" is a retention killer. New viewers don't care. Start with the Hook immediately (0:00).
2. The "Desktop Recording" Snooze: Showing a static screen for >10 seconds. AI can read it, but humans get bored. Use Pattern Interrupts (zoom, highlight, face cam) every 5-10 seconds.
3. The "Premature Goodbye": Saying "In summary..." or "Thanks for watching." This signals the video is over, and viewers leave before the CTA. Use the Fortune Cookie transition instead ("The real insight is...").
TRACKS (Paths to Authority)
1. The "WSA Repurpose" Workflow: Take a Module 3 Article -> Extract the 7 Sections -> Map to Video Chapters -> Record. This ensures you have deep, structured content ready to film.
2. The "Series" Strategy: Don't just make one video. Make a 3-part series (Part 1: Problem, Part 2: Mechanism, Part 3: Case Study). This builds Topical Authority and binge sessions.
STACKS (Tool Combos)

Output: Production-ready video scripts, SEO metadata (Title, Description, Schema), and Thumbnail concepts.
B. Inputs Required

D. Step-by-Step Workflow (6 Phases - Detailed Breakdown)
Total Timeline: 16 hours over 3 days Team: 1 strategist/writer (can be same person) Prerequisites: Module 3 WSA article completed, Module 2 BOFU keyword selected
Phase 1: Pre-Production Foundation (Day 1 Morning - 3 hours)
Goal: Establish psychological foundation and structural framework before writing a single word.
• Step 1.1: Load Inputs from Module 3 (30 minutes)
    ? Action 1: Import WSA Article. Open the Module 3 WSA article and extract the 7 core sections (Safety Anchor, Mechanism, Proof, Action Path, Objection Kill, Value Stack, Closer). Copy into a new document titled "Video Source Material".
    ? Action 2: Identify Core Insights. Read through WSA and highlight the 3 most surprising/valuable insights. Ask: "What made me go 'Whoa, I didn't know that' when researching this?" Mark these as "EVR Candidates".
    ? Action 3: Extract BOFU Keyword Data. Pull primary keyword from Module 2. Note search volume, KD, current ranking, and semantic variations.
• Step 1.2: EVR Psychology Planning (45 minutes)
    ? Action 1: Define Current Expectation. What does your target avatar currently believe about this topic? What is the "safe" conventional wisdom they've been taught?
    ? Action 2: Craft the Reality Contrast. What data/experience contradicts this expectation? What is the surprising truth? Can you quantify the gap?
    ? Action 3: Score the EVR Gap. Use the 0-10 scale. Test on 2-3 people in target audience. Target score: 7+.
    ? Quality Gate: If score <7, iterate on the reality contrast. Add more specificity or data.
• Step 1.3: Story Archetype Selection (30 minutes)
    ? Action 1: Review the 12 Archetypes (Comparison, Tutorial, Contrarian, Case Study, Listicle, etc.).
    ? Action 2: Match Archetype to Content + Avatar Stage.
        ? IF avatar is comparing solutions ? Use Comparison.
        ? IF avatar knows the problem but not the fix ? Use Tutorial or Problem ? Solution.
        ? IF avatar believes conventional wisdom ? Use Contrarian.
    ? Action 3: Select Primary and Backup Archetype.
• Step 1.4: Neuro-Axis Video Mapping (45 minutes)
    ? Action 1: Create Timeline Grid. Map minutes to Neuro-Axes (BODY, HEART, MIND, SPIRIT, PSYCHE, CONNECTION).
    ? Action 2: Map WSA Sections to Timeline (e.g., WSA Safety Anchor ? Video Minute 0-1 BODY axis).
    ? Action 3: Check Excitatory/Inhibitory Balance. Calculate difference between Excitatory (HEART, PSYCHE, CONNECTION) and Inhibitory (BODY, MIND, SPIRIT). Target difference: ?4 points.
    ? Note: ? Refer to: NEURO_RESONANCE_AUDITOR_v2_1.docx for complete axis definitions.
• Step 1.5: Format Decision (30 minutes)
    ? Action 1: Determine Video Length based on archetype complexity (e.g., Comparison = 8-12 mins).
    ? Action 2: Determine Visual Style (Face cam, screen share, B-roll).
    ? Action 3: Decide on Production Complexity Tier (Tier 1: Simple, Tier 2: Standard, Tier 3: Advanced).
Phase 2: Hook Engineering (Day 1 Afternoon - 2.5 hours)
Goal: Create 20 hook variations, select top 3, design thumbnail, verify 3-hook alignment.
• Step 2.1: Generate 20 Hook Variations (60 minutes)
    ? Action 1: Use the 20 Hook Templates (from KB-VIRAL-HOOK-ENGINEERING). Select 10 templates matching your EVR gap.
    ? Action 2: Create Archetype-Specific Hooks.
    ? Action 3: Add Quantification. Every hook needs a number (time, money, result, data point).
    ? Reference: ? Refer to: VIRAL RESONANCE ARCHITECT v1.docx for hook psychology.
• Step 2.2: Hook Quality Audit (30 minutes)
    ? Action 1: Run 8-Point Audit on Each Hook (Single Subject? Single Question? EVR 7+? BOFU Intent?).
    ? Action 2: Select Top 3 Hooks. Pick highest scoring as Primary, 2 runners-up as Variants.
• Step 2.3: Thumbnail Storyboarding (60 minutes)
    ? Action 1: Select Stun Type (Person, Recognizable, Atypical Visual, Atypical Layout).
    ? Action 2: Create Thumbnail Concept using formula: [RECOGNIZABLE ELEMENT] + [EMOTION/DATA] + [TEXT OVERLAY].
    ? Action 3: Verify 3-Hook Alignment. Ensure Visual (Thumbnail), Text (Title), and Spoken (First 15s) all promise the exact same outcome.
Phase 3: Script Outline (Day 2 Morning - 2 hours)
Goal: Build skeletal structure with chapters, connectors, and uniqueness validation.
• Step 3.1: Uniqueness Gate Check (30 minutes)
    ? Action 1: Search Competitor Videos. Watch top 3 results for your keyword.
    ? Action 2: Find Your Unique Angle (New Information, New Delivery, New Phrasing, New Examples).
    ? Gate Decision: If outline is too similar to competitors, FAIL. Go back to research.
• Step 3.2: Build Chapter Outline (60 minutes)
    ? Action 1: Define 5-7 Main Chapters. Use keyword-rich chapter naming (e.g., "What are BOFU keywords?" instead of "Introduction").
    ? Action 2: Map Chapters to Neuro-Axis.
    ? Action 3: Add What/Why/How for Each Chapter.
• Step 3.3: Grease-Slide Connector Planning (30 minutes)
    ? Action 1: Identify Transition Points.
    ? Action 2: Select Grease-Slide Phrases (e.g., "But here's the part that shocked me...", "Which brings us to the exact process...").
    ? Action 3: Add Rehook Notes. Schedule a rehook every 90-120 seconds.
Phase 4: Script Writing (Day 2 Afternoon - 4 hours)
Goal: Write full script in correct order (Body ? Intro ? Outro), add visual cues.
• Step 4.1: Write Body FIRST (2 hours)
    ? Principle: You can't write a compelling hook until you know what you're hooking people into.
    ? Action 1: Expand Chapter Outlines into Full Script. Write conversationally.
    ? Action 2: Add B-Roll and Visual Notes (Visual, On-screen text, Graphics). Rule: Change visual every 5-10 seconds.
    ? Action 3: Insert Data Tables and Citation Magnets.
• Step 4.2: Write Intro SECOND (60 minutes)
    ? Action 1: Write Hook Using EVR Formula: Pattern Interrupt + EVR Statement + Credibility Anchor + Promise + Curiosity Gap.
    ? Action 2: Verify 3-Hook Alignment.
    ? Action 3: Add Thesis Statement.
• Step 4.3: Write Outro LAST (45 minutes)
    ? Action 1: Write Fortune Cookie Insight (Zoom out to meta-lesson).
    ? Action 2: Write Native Embed CTA using formula: [Identify natural next step] + [Position resource as solution] + [Clear instruction].
    ? Action 3: Add Loop to Next Video.
• Step 4.4: Add Visual Cues and B-Roll Notes (15 minutes)
    ? Action: Review full script for visual gaps. Add pattern interrupt schedule every 45-60 seconds (Sound effect, zoom, color flash).

Goal: Transform AI-sounding script into natural, conversational spoken language.

    ? Pass 4: Contraction Addition. (It is ? It's).

    ? Pass 6: Sensory Language Addition. Add 1-2 sensory words per paragraph.

    ? Action 1: Generate VideoObject Schema (JSON-LD).
    ? Action 2: Add FAQPage Schema.
E. Quality Checks (The "Gate")
• [ ] The "Kitchen Table Test": Did you read the script aloud? Does it sound like a conversation?
• [ ] The "3-Hook Alignment": Do thumbnail, title, and first sentence promise the exact same outcome?
• [ ] The "EVR Score": Does the hook create a gap >7/10?
• [ ] The "Citation Check": Are there named frameworks or data tables on screen for AI to read?
F. Failure Modes (Diagnostics & Recovery)
Failure Mode 1: The "Boring Hook" Death Spiral
• Symptom: CTR <3% or 40% drop-off in first 30 seconds. Impressions are high but views are low.
• Root Cause: Hook confirms expectation instead of creating an EVR gap.
• Diagnostic Questions: What is the EVR score? Is it <7? Do the 3 hooks align?
• Recovery:
    1. Calculate current EVR score.
    2. Find the hidden EVR (most surprising fact buried in video).
    3. Rewrite hook using 20 Templates.
    4. Create new thumbnail featuring data point.
    5. A/B test new title/thumbnail.
    6. Add pattern interrupt at 0:05.
