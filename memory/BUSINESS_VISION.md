## Regulatory Framework (Embedded Guardrails)
- **CAN-SPAM**: Unsubscribe link, physical address, honest subjects → M5 Guardian Agent
- **TCPA**: Prior express consent for AI voice; no cold robocalls → Speed-to-lead <5 min
- **GDPR/CCPA**: Centralized Private Lead Database (Supabase), consent tracking, right to erasure
- **Opt-Out Detection**: Explicit keywords + implicit hostile sentiment → DNC propagation
- **Blocklist Propagation**: Master Blocklist synced cross-channel (email + LinkedIn + voice)
- **Content Safety**: No financial/medical/legal advice → Guardian Agent blocks + rewrites
- **Volume Controls**: 30-50 emails/inbox/day, spam <0.1%, LinkedIn throttled to interested replies
